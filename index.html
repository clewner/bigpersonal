Theme Tracker

I developed Theme Tracker, a web app that lets students track specific words and phrases across any book in the public domain, returning every paragraph that has one of the words they choose. This allows for a better understanding of recurring themes and symbols in the text, and can help when writing analytical essays that require text evidence.

  • Conceptualized, designed, and developed Theme Tracker.
  • Utilized Python, Flask, HTML, and JavaScript to create a user-friendly interface and efficient back-end functionalities.
  • Developed a web scraper to dynamically access and analyze books from public domain databases, eliminating the need for manual downloads.
  • Hosted the website on a free server using PythonAnywhere, managing all aspects of deployment and maintenance.
  • Incorporated user feedback to continuously improve website features, enhancing user satisfaction and engagement.
  • Served over 100 unique users.
